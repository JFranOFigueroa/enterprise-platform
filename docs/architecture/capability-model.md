# Enterprise Platform - Capability Model

## 5 Layers

1. **Experience Layer** - How developers interact
2. **Delivery Layer** - Git → CI → Registry → GitOps
3. **Platform Layer** - Runtime, Networking, Observability, Scaling
4. **Data Layer** - PostgreSQL, Storage, Backups
5. **Infrastructure Layer** - Nodes, CPU, Memory, Network
